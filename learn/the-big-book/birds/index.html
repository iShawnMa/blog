Bird family tree#

species / ˈspiːʃiːz /

（动植物的）种，物种；种类；（具有相同特征的）一类人；（化学，物理）核素；（圣餐中用的）面包和圣酒

birds of prey

猛禽：指捕食其他动物的鸟类，如鹰、鹫、隼等。

prey / preɪ /

n.猎物，捕获物；受害者，受骗者；<古>掠夺品 v.捕食；欺凌，坑害（尤指弱者）；不断困扰，折磨

hawk / hɔːk /

n.鹰，隼；鹰派分子，主战分子；（似鹰般）贪婪的人；灰泥托板 v.叫卖，兜售；咳痰；（人）带鹰出猎；（鸟，蜻蜓）飞行觅食

vulture / ˈvʌltʃər /

n.秃鹫，兀鹫；趁火打劫的人，贪婪的人

falcon / ˈfælkən /

n.隼，猎鹰

talon / ˈtælən /

n.爪；[建]爪饰；如爪的手

woodland / ˈwʊdlənd /

n.林地；森林 adj.林地的；居住在森林中的

woodpecker / ˈwʊdpekər /

n.[鸟] 啄木鸟

toucan / ˈtuːkæn /

n.犀鸟，[鸟] 巨嘴鸟；巨嘴鸟座

parrot / ˈpærət /

n.鹦鹉；学舌者，机械模仿别人的人 v.机械地模仿，鹦鹉学舌般重复

hummingbird / ˈhʌmɪŋbɜːrd /

n.蜂鸟

peacock / ˈpiːkɑːk /

n.（雄）孔雀；注重仪表和衣饰的男子，虚荣骄傲的男子 v.炫耀，神气活现地走

pigeon / ˈpɪdʒɪn /

n.鸽子；<非正式，美>（容易）上当受骗的人；<英>（自己的）责任，分内之事；<俚>己方飞机

perching birds

雀形目，栖离

perch / pɜːrtʃ /

v.（鸟）栖息，停留；（尤指人在高处或很窄的地方）稍坐，暂歇；（建筑物）坐落于……的上方（或边缘）；把……平稳置于，使坐落于 n.（鸟的）栖木，栖枝；高处，高位；河鲈，鲈鱼；杆（尤指用来测量土地长度的单位）

sparrow / ˈspæroʊ /

n.麻雀；（任何在体积或颜色方面）像麻雀的鸟类

robin / ˈrɑːbɪn /

n.欧亚鸲，欧洲知更鸟；旅鸫，美洲知更鸟（毛灰色，胸部红色，比欧亚鸲大）

thrush / θrʌʃ / n.画眉；[口腔]鹅口疮；蹄叉腐疽

swallow / ˈswɑːloʊ /

v.吞下，咽下；（由于紧张等）做吞咽动作，咽口水；吞没，淹没；耗尽（金钱，资源）；完全相信，信以为真；抑制（感情、言语）；默默忍受（侮辱、批评等） n.吞，一口；燕子

crow / kroʊ /

n.乌鸦；（公鸡或像公鸡的）啼叫声；克劳人（分布在美国西部的美洲土著居民）；<非正式>妇人，丑妇 v.欢叫，高兴地说；自鸣得意

finch / fɪntʃ /

n.雀科鸣鸟；雀类

magpie / ˈmæɡpaɪ /

n.鹊，喜鹊；饶舌的人；有收集零碎东西癖好的人 adj.鹊的；有收集癖的；斑驳的，混杂的

white wagtail / ˈwæɡteɪl /

白鹡鸰

grip / ɡrɪp /

n.紧握；掌握，控制；理解；不打滑，走得稳；手柄；紧握（或抓牢）的方式；（拍摄电影时）摄影机和照明设备管理人员；<英>发夹；旅行袋 v.握紧，夹住；（尤指通过摩擦力）牢牢地附着，紧贴；使感兴趣，吸引住（某人）的注意；对……产生强有力的影响

puffin / ˈpʌfɪn /

n.善知鸟；海鹦；海鸭之类

albatross / ˈælbətrɑːs /

n.[鸟] 信天翁；沉重负担

pelican / ˈpelɪkən /

n.[鸟] 鹈鹕

cormorant / ˈkɔːrmərənt /

n.鸬鹚；贪婪的人 adj.贪婪的

feather / ˈfeðər /

n.羽毛 v.用羽毛装饰；长羽毛；使（螺旋桨）顺流交距；使（旋翼）周期变距；（划船）平掠回（桨）；如羽毛般浮动（或移动、飘动）；小心翼翼地把……混合起来（或把……弄平）；（油墨、口红等抹上后）化开，变模糊

webbed feet

蹼足：一种动物足部特征，指趾间具有薄膜的足，通常用于水生动物，如鸭子、青蛙等，有助于游泳。

goose / ɡuːs /

n.鹅；鹅肉；雌鹅；<非正式> 傻瓜；（裁缝用的）熨斗 v.<美，非正式> 触摸（或捏）某人的臀部；<美，非正式> 推动，激励；<非正式>开大（汽车等）的油门；<非正式>作嘘嘘声反对（某人）

swan / swɑːn /

n.天鹅；天鹅星座（Swan） v.<英，非正式>无目的地漫游，闲逛

heron / ˈherən /

n.鹭，苍鹭；深紫灰

avocet / ˈævəset /

n.反嘴鹬；反嘴鹬科

flamingo / fləˈmɪŋɡoʊ /

n.[鸟] 火烈鸟

ostrich / ˈɑːstrɪtʃ /

n.鸵鸟；逃避现实的人，不愿正视现实者

emu / ˈiːmjuː /

n.鸸鹋（产于澳洲的一种体型大而不会飞的鸟）

takahe / ˈtɑːkəhiː /

南秧鸟 巨水鸡

kiwi / ˈkiːwi /

n.<非正式>新西兰人（Kiwi）；几维（新西兰鸟，不能飞）；奇异果

I’m a bird spotter#

clue / kluː /

n.线索，提示；理解，想法

v.给（某人）提供线索，使（某人）了解

poo / puː /

n.（儿语）屎；拉屎

v.排便，拉屎；表示厌恶地呸；表示不屑地嘁

regurgitate / rɪˈɡɜːrdʒɪteɪt /

vi.回流；回涌；反胃；反刍

vt.使回流；使回涌；使反刍

binocular / bɪˈnɑːkjələr /

adj.[生物] 双眼的；双目并用的

n.双筒望远镜

sketchbook / ˈsketʃbʊk /

n.小品集；写生簿；随笔集
